# Alvaros-Project

Este projeto é um sistema de gerenciamento para academias, desenvolvido para ajudar no controle de alunos, planos de treino, acompanhamento de evolução e avisos de vencimento de planos.

## Funcionalidades

- **Cadastro de Alunos:** Adicionar, editar e gerenciar informações dos alunos.
- **Gestão de Planos de Treino:** Permitir que professores montem e editem planos de treino personalizados.
- **Acompanhamento de Evolução:** Registrar e visualizar o progresso dos alunos, como peso e medidas.
- **Alertas de Vencimento:** Notificar sobre a proximidade do vencimento dos planos dos alunos.

## Tecnologias Utilizadas

- **Backend:** PHP Orientado a Objetos (OOP) e SQL (mysql)
- **Frontend:** HTML, CSS e JavaScript

## Como Executar o Projeto

Siga os passos abaixo para rodar o projeto em sua máquina local:

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/](https://github.com/)[SeuUsuario]/Alvaros-Project.git
    ```

2.  **Navegue até a pasta do projeto:**
    ```bash
    cd Alvaros-Project
    ```

3.  **Inicie o servidor local do PHP:**
    Se você já tem o PHP instalado, pode usar o servidor embutido para testar. No terminal, execute o seguinte comando:
    ```bash
    php -S localhost:8000
    ```

4.  **Acesse o sistema:**
    Abra o seu navegador e acesse a URL: `http://localhost:8000`
